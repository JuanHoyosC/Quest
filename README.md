# 🛠 Cronograma de Desarrollo del MVP (6 semanas)  

📌 **Objetivo:** Implementar autenticación, creación de salas, preguntas con 6 respuestas máximas y visualización de partidas.  

---

## 🚀 Semana 1: Configuración del Proyecto y Autenticación  
- [x] Configurar entorno de desarrollo (**Angular + Ionic + Amplify o Firebase**)  
- [x] Implementar autenticación (**registro/inicio de sesión con email y Google**)  
- [x] Gestionar **roles** (profesor y estudiante)  
- [x] Redirigir usuarios según su rol  

---

## 🚀 Semana 2: Creación de Salas y Modelo de Datos  
- [x] Definir **estructura en la base de datos** (DynamoDB o Firestore)  
- [x] Permitir a los profesores **crear salas**  
- [x] Asignar **código único** a cada sala  
- [x] Implementar **listado de salas** creadas por cada profesor  

---

## 🚀 Semana 3: Gestión de Preguntas y Respuestas  
- [x] Permitir agregar **preguntas con hasta 6 respuestas**  
- [x] Guardar preguntas en la base de datos  
- [x] Validar que **al menos una respuesta sea correcta**  
- [x] Asignar preguntas a una sala específica  

---

## 🚀 Semana 4: Unirse a Partidas y Flujo del Juego  
- [x] Estudiantes pueden **ingresar código de sala**  
- [x] Validar **código de sala antes de unirse**  
- [x] Mostrar **lista de jugadores en la sala**  
- [x] Mostrar preguntas a los jugadores cuando inicia la partida  

---

## 🚀 Semana 5: Visualización de Resultados y Experiencia de Usuario  
- [x] Mostrar **respuestas de los estudiantes en tiempo real**  
- [x] Guardar **estadísticas de la partida**  
- [x] Mostrar al profesor el **resumen de la partida** (quién acertó más, tiempos de respuesta)  
- [x] Mejorar **diseño y experiencia de usuario**  

---

## 🚀 Semana 6: Pruebas, Optimización y MVP Finalizado  
- [x] **Pruebas de funcionalidad** en distintos dispositivos  
- [x] Corrección de **errores y optimización de rendimiento**  
- [x] **Desplegar MVP** en una plataforma de pruebas (ej. Firebase Hosting)  
- [x] Recopilar **feedback inicial**  

---

## 🚀 Futuras Iteraciones Después del MVP  
- **Sistema de pagos:** Monetización a través de suscripciones o pagos únicos.  
- **Rankings y gamificación:** Tablas de líderes y recompensas por participación.  
- **Modos de juego adicionales:** Preguntas cronometradas, desafíos entre jugadores, etc.  
- **Reportes avanzados:** Análisis de desempeño de los estudiantes.  
- **Soporte para múltiples idiomas:** Internacionalización de la aplicación.  
- **Integración con IA:** Generación de preguntas automáticas con IA.
